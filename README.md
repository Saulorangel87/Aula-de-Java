# 🧬 BioFit Pro - Calculadora de Composição Corporal

![Status do Projeto](https://img.shields.io)
![Vercel](https://img.shields.io)

**BioFit Pro** é uma aplicação web responsiva projetada para profissionais de educação física e entusiastas da saúde. Ela permite realizar avaliações físicas precisas utilizando o **Protocolo de Jackson-Pollock (3 Dobras)**, fornecendo dados detalhados sobre massa magra, massa gorda e IMC.

---

## 🚀 Funcionalidades

- **Cálculo de Composição Corporal:** Algoritmo baseado em dobras cutâneas (Jackson-Pollock).
- **Interface Dinâmica:** Labels que se ajustam automaticamente ao sexo selecionado (Peito/Abdômen vs. Braço/Cintura).
- **Visualização de Dados:** Gráficos interativos tipo "Doughnut" que ilustram a divisão de massa corporal.
- **Relatório Profissional:** Função integrada para exportação de resultados em formato PDF.
- **Responsividade:** Design otimizado para uso em dispositivos móveis dentro da academia.

## 🛠️ Tecnologias Utilizadas

- **HTML5 & CSS3:** Estruturação e estilização moderna (Flexbox/Grid).
- **JavaScript (ES6+):** Lógica de cálculo e manipulação do DOM.
- **[Chart.js](https://www.chartjs.org):** Renderização de gráficos dinâmicos.
- **[html2pdf.js](https://ekoopmans.github.io):** Geração de documentos PDF no lado do cliente.

## 📐 O Protocolo Utilizado

A aplicação utiliza o protocolo de **Jackson-Pollock 3 dobras** combinado com a **Equação de Siri**. As medidas de entrada devem ser inseridas em milímetros (mm) para garantir a precisão do cálculo da densidade corporal.
